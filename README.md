# Brainfuck
Brainfuck interpreter in C++

## Requirements
* C++ compiler

## Usage
Compile `main.cpp` inside `src/` folder and run as `<program> <file>`

For example:
```
cd src
gcc main.cpp -o interpreter  
./interpreter ../ex/hello.b
```
Should generate
`Hello World!`
as its output.
